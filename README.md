# 016ssm物业管理系统
016ssm物业管理系统


## 功能介绍
物业管理系统，包含登录信息统计、物业人员管理，住户管理，房屋管理、车位管理等功能；


源码获取： [**点此获取**](http://www.shuyue.fun/index.php?type=productinfo&id=119)

## 环境要求

JDK >= 1.8 (推荐1.8版本)

Mysql >= 5.5.0 (推荐5.7版本)

Maven >= 3.0

开发工具：IDEA/Eclipse

技术栈：后端Spring+Spring MVC+MyBatis 前端：html+JQuery

Tomcat: 8.0及以上
